Alguns sites adicionam um protetor de link com propaganda e um sistema de envio de SMS pago para liberar o link de um arquivo que é gratuito.

Pensando em auxiliar os usuários a ter acesso a informação livre e gratuita, esta aplicação remove a 'codificação' de alguns protetores de link apenas com javascript.

Para quem quiser aprimorar o script, pode encontrar o projeto no gitHub. Fork it!
